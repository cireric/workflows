# 非虚构类（历史/经济/科学/哲学/方法类）

## 阅读策略

非虚构的核心是论证质量。引导时关注：

- **方法论局限**：作者的结论依赖什么方法？这个方法能支撑这个结论吗？
- **因果 vs 相关**：作者说X导致Y——是真的因果，还是只是相关？有没有替代解释？
- **样本与偏差**：论据来自什么样本？有没有选择偏差？反例在哪？
- **隐含假设**：论证中没明说但必须成立的前提是什么？如果前提不成立呢？
- **可证伪性**：作者的观点能被反驳吗？他自己考虑了哪些反证？

## A 理解确认

**阶段目标**：确认用户是否准确把握书的核心论点和论证逻辑

**提问方向**（根据用户回答选择，不固定顺序，每次选 1 个方向深入）：

- **核心论点**：这本书的主张是什么？作者在试图解决什么问题？
- **论据结构**：作者用了哪些论据？最有说服力的是哪个？最弱的呢？
- **论证逻辑**：能用自己的话复述作者的推理链条吗？

**追问示例**：

- 用户说"作者认为X导致Y" → "他是怎么证明因果关系的？有没有可能只是相关？"
- 用户说"论证很有说服力" → "你觉得哪个论据最有力量？反过来，哪个论据你觉得最站不住？"
- 用户说"核心观点是Z" → "如果用一句话反驳这个观点，你会怎么说？"

**备选问题**（LLM 不熟悉该书时可用）：

- 这本书的核心观点/主张是什么？作者在试图解决什么问题？
- 作者用了哪些论据来支撑这个观点？你觉得最有说服力的是哪一个？
- 你能用自己的话复述一下作者的论证逻辑吗？作者的叙事方式（如果有的）对理解有什么影响？

**进入 B 阶段条件**：用户能准确复述核心论点，且理解论证的基本结构

## B 批判分析

**阶段目标**：引导用户审视论证质量、发现隐含假设和对立观点

**提问方向**：

- **隐含假设**：论证中有没有没明说但必须成立的前提？如果前提不成立呢？
- **视角局限**：作者的立场可能忽略了什么？有没有选择偏差？
- **对立观点**：对于这个问题，有没有其他合理的解释？作者是如何回应的？

**追问示例**：

- 用户说"论证很严密" → "有没有什么前提是作者没有论证就直接假设成立的？"
- 用户说"作者没考虑X" → "如果考虑了X，作者的结论会变吗？还是会更强？"

**备选问题**（LLM 不熟悉该书时可用）：

- 作者的论证中有没有隐藏的假设？他的视角可能忽略了什么？
- 对于这个问题，有没有其他合理的解释或对立观点？作者是如何回应的？

**进入 C 阶段条件**：用户能指出论证的至少一个薄弱环节或隐含假设

## C 个人共鸣

**阶段目标**：引导用户将书中论点与自身认知和经验对照

**提问方向**：

- **认知改变**：这本书改变了你对这个议题的哪些既有认知？
- **经验印证/冲突**：你的个人经历是支持还是反驳作者的观点？
- **行为改变**：你会因为这本书而改变什么实际行为或决策方式吗？

**追问示例**：

- 用户说"改变了我的看法" → "具体从什么看法变成了什么？是什么论据让你转变的？"
- 用户说"和我的经验一致" → "能举个例子吗？你的经验能不能用作者的理论解释？"

**备选问题**（LLM 不熟悉该书时可用）：

- 这本书改变了你对这个议题的哪些既有认知？
- 你会因为这本书而改变什么实际行为或决策方式吗？

**进入 D 阶段条件**：用户建立了书中论点与自身经验/认知的连接

## D 超越书本

**阶段目标**：引导用户将认知变化延伸到更广的领域或更深的行动

**提问方向**：

- **认知迁移**：这本书改变了你的什么看法？这种改变会影响你未来的什么选择？
- **回溯反思**：重新回答 A 阶段的问题，你的答案改变了吗？是什么让你改变？
- **延伸应用**：这个论点/方法可以应用到其他什么领域？边界在哪？

**追问示例**：

- 用户说"让我更理性了" → "具体在什么场景下你会用书中的方法来替代你原来的直觉？"
- 用户说"没什么实际改变" → "如果书中的论点在你身边真实发生了，你觉得你会怎么应对？"

**备选问题**（LLM 不熟悉该书时可用）：

- 这本书带给你的认知变化，会如何影响你未来的思考或行动？
  → 追问方向：重新回答第一阶段的问题，你的答案改变了吗？是什么让你改变？
