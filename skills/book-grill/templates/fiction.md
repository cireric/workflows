# 虚构文学（小说/科幻/文学/戏剧/诗歌）

## 阅读策略

小说的深层意义藏在叙事技术里。引导时关注：

- **叙事视角**：谁在讲故事？视角切换意味着什么？不可靠叙述者是否在误导读者？
- **时间线操控**：倒叙、插叙、多线并行——作者为什么这样安排而非按时间顺序？
- **人物动机**：角色说自己为什么做，和真正原因可能不同——追问"他嘴上说的和实际做的之间有什么差距？"
- **留白与象征**：未明说的往往比明说的更重要——追问"作者在这里故意没写的是什么？"

## A 理解确认

**阶段目标**：确认用户是否准确把握故事的核心内容与叙事结构

**提问方向**（根据用户回答选择，不固定顺序，每次选 1 个方向深入）：

- **情节线**：主要故事是什么？有没有特别意外的支线？
- **人物**：主人公的决策轨迹你认同吗？有没有不合理的行为？嘴上说的和实际做的一致吗？
- **叙事**：作者在叙事手法上有什么特别之处？（视角切换/时间线/语言风格/不可靠叙述者）

**追问示例**：

- 用户说"情节很复杂" → "能具体说说哪条线让你觉得复杂吗？它和主线是什么关系？"
- 用户说"主人公很蠢" → "哪个决策让你这么觉得？作者为什么要这样写——是在暗示什么吗？"
- 用户说"叙事手法很特别" → "你觉得作者为什么要选择这种视角来讲故事？如果换一个视角会怎样？"

**备选问题**（LLM 不熟悉该书时可用）：

- 故事的主要情节线是什么？有没有哪条支线让你觉得特别意外？
- 主人公的决策和成长轨迹你认同吗？有没有什么行为让你觉得不合理？
- 作者在叙事手法上有什么特别之处？（视角切换/时间线/语言风格）

**进入 B 阶段条件**：用户能准确描述核心情节，且对人物动机或叙事手法有基本理解

## B 批判分析

**阶段目标**：引导用户挖掘文本的深层主题和作者意图

**提问方向**：

- **深层主题**：这个故事在探讨什么？通过情节传达了哪些关于人性/社会的观点？
- **作者意图 vs 读者感受**：作者想表达的和你感受到的一致吗？有没有"说一套做一套"的地方？
- **替代可能**：如果换一个不同的结局/视角/设定，故事的核心信息会变吗？

**追问示例**：

- 用户说"这本书讲的是人性" → "人性是个很大的词——书里具体哪个情节让你看到了什么样的人性？"
- 用户说"结局很震撼" → "你觉得作者选择这个结局是在说什么？如果结局不同，作者想表达的东西还成立吗？"

**备选问题**（LLM 不熟悉该书时可用）：

- 这个故事在探讨什么更深层的主题？它通过情节传达了哪些关于人性/社会的观点？
- 如果换一个不同的结局，你觉得故事的核心信息会变吗？

**进入 C 阶段条件**：用户能超越字面理解，触及主题层面

## C 个人共鸣

**阶段目标**：帮助用户发现书中内容与自身经历/情感的连接

**提问方向**：

- **人物投射**：有没有哪个角色让你联想到自己或身边的人？
- **经验共振**：书中的什么情境让你想到了自己的某段经历？
- **认知触发**：读完之后，对哪些以前没想过的问题开始有了自己的看法？

**追问示例**：

- 用户说"我很有共鸣" → "能具体说说吗？是哪个情节或哪个人物触发了你？"
- 用户说"没什么共鸣" → "这本书让你觉得遥远，是因为它的世界和你生活的世界差距太大吗？还是因为作者的处理方式让你无法代入？"

**备选问题**（LLM 不熟悉该书时可用）：

- 故事中有没有哪个人物让你联想到自己认识的人？或者让你想到自己的某段经历？
- 读完这个故事，你对哪些以前没想过的问题开始有了自己的看法？

**进入 D 阶段条件**：用户建立了至少一个书中内容与自身的连接

## D 超越书本

**阶段目标**：引导用户将阅读收获延伸到更广的思考或行动

**提问方向**：

- **认知迁移**：这本书改变了你的什么看法？这种改变会影响你未来的什么选择？
- **普遍化**：如果这个启示被广泛应用，世界会因此变得更好还是更坏？
- **行动指引**：你会因为这本书做任何不同的事吗？

**追问示例**：

- 用户说"让我思考了很多" → "能举个具体的例子吗？下次遇到类似情况你会怎么做？"
- 用户说"没什么实际行动的改变" → "如果这本书的某个观点在你生活中真的发生了，你觉得你会怎么应对？"

**备选问题**（LLM 不熟悉该书时可用）：

- 这本书带给你的认知变化，会如何影响你未来的思考或行动？
  → 追问方向：如果这个启示被广泛应用，世界会因此变得更好还是更坏？
