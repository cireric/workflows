# Researcher Agent Prompt

Agent 系统提示词的设计与优化——以 agents/researcher.md 为核心对象。

## Language

**门控 (Gate)**:
阶段转换时强制执行的填表式审计协议。LLM 的角色是填充答案而非选择攻击方向。
_Avoid_: 检查点 (checkpoint)、审查 (review)

**轻量输出**:
简单查询在标准输出结构中的自然收敛——Hypotheses/Risks 等结构存在但内容精简。不是审计协议的简化，不是可选择的输出模式。
_Avoid_: 轻量模式、简化输出、省略结构

**意图级表述**:
在概述或引用点保留规则的设计意图和方向，不重复操作细节。操作细节指向唯一权威定义点。
_Avoid_: 摘要、缩写

**元认知检查**:
研究者对自身思维过程的审视——锚定检测、复杂化检测、确信度校准。与问题框架级检查（审视外部对象）正交。
_Avoid_: 自省、反思

**回溯依赖**:
模型执行某条规则时，需要跳转到远处查阅完整定义的依赖关系。空间距离越大，注意力衰减越严重，依赖越可能失效。
_Avoid_: 跨节引用、前向引用（过于技术化，不精确）

**停止信号**:
搜索扩展的终止条件。仅适用于中/低重要性结论；高重要性结论的停止条件见 EXHAUSTIVE SEARCH。
_Avoid_: 搜索终止、搜索上限
